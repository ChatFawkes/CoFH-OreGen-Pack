CoFH-Oregen-Pack
================
This is a set of configs that adds ores to CoFH World generation. All files from other mods are left default and only have their respective ores disabled.

##Ores Supported:
- Copper
- Tin
- Silver
- Lead
- Nickel/Ferrous
- Aluminum
- Zinc
- Osmium
- Certus Quartz
- Ruby
- Sapphire
- Amethyst
- Peridot
- Vanilla Ores

##Planned support:
- Uranium
- Teslatite
- Tungsten

(Might do more upon request)
